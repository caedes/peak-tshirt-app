# Peak Tshirt Web App

> On vend des tshirts et on le fait bien.

## Getting Started

Install dependencies:

```shell
yarn
```

Run the dev server and the local API:

```shell
yarn start
yarn start:api
```

Make your move, develop your new feature, then commit with:

```shell
yarn commit
```
